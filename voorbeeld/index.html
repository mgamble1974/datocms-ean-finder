EAN Zoeker — AI prototype

Twee AI-opties om productgegevens binnen te halen:
  • Optie B — OpenAI web search, gefocust op Bol.com, Coolblue, MediaMarkt en fabrikantensites. Veiliger en sneller, maar afhankelijk van wat OpenAI kan vinden.
  • Optie C — Server haalt zelf de Bol.com zoekpagina op, volgt de eerste productlink, en stuurt de HTML naar GPT-4o voor extractie. Grotere kans op data, maar Bol.com kan blokkeren.
⚠️ Prototype — Optie C scrapet Bol.com server-side. Niet geschikt voor productie/publiek gebruik. Voor DatoCMS-plugin bouwen we hier later op door.
